# Maddex

Marketing website for Maddex — a professional financial intelligence terminal for Australian investors.

Built with React, Vite, Tailwind CSS, Framer Motion, and React Router.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Pages

- `/` — Home
- `/product` — Terminal modules
- `/maddenai` — MaddenAI scoring engine
- `/pricing` — Plans and pricing
- `/about` — Company and founder
- `/research` — Madden Research newsletter
